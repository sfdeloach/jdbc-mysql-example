# jdbc-mysql-example
Database exports made easy

Library Dependencies:
 - gson-2.8.5.jar
 - mysql-connector-java-8.0.13.jar

Must contain ./connection-data/connection.json:

```
{
  "url": "jdbc:mysql://****",
  "database": "****",
  "dbUser": "****",
  "password": "****",
  "query": "SELECT * FROM ****"
}
```
